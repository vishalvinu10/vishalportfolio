Step 2: Profile Image (Optional)

Current Status: Using a placeholder with gradient background

To Add Real Image: Save as vishal-profile.jpg

Recommended Size: 400x400 pixels or larger (square format)

Format: JPG, PNG, or WebP

Step 3: How to Add Images

  1. Save your profile image as vishal-profile.jpg
  2. Place it in the same folder as index.html
  3. The image will automatically appear in the hero section

Step 4: Alternative Image Names

If you want to use different image names, update the HTML file:

<img src="your-image-name.jpg" alt="Vishal M - QA Test Engineer" class="profile-img">

Step 5: Image Optimization Tips

  • Compress images to reduce file size
  • Use WebP format for better performance
  • Keep file size under 500KB for faster loading
  • Use a professional headshot or business photo
